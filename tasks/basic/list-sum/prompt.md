Write a function with this exact signature:

    fn list_sum(xs: List[Int]) -> Int

Behavior:

- Return the sum of all elements in `xs`
- `list_sum([])` must return `0`
- `list_sum([1, 2, 3])` must return `6`

You may use `list.fold`, recursion, or `list.sum`.
Note: if you use `list.sum`, the function body is trivially `list.sum(xs)`.
The test is whether you can write a correct function signature and body in Almide.
