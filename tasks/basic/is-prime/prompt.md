Write a function with this exact signature:

    fn is_prime(n: Int) -> Bool

Behavior:

- Return `true` if `n` is a prime number, `false` otherwise
- A prime is a natural number greater than 1 whose only positive divisors are 1 and itself
- `is_prime(0)` and `is_prime(1)` must return `false`
- Negative inputs must return `false`
